Features

Everything stays local

1Password Integration

Fetches credentials via official 1Password CLI. No external servers. Your passwords never leave your machine.

Automatic MFA

TOTP from 1Password. SMS from Messages. Email from Gmail. Handles 2FA automatically so you don't have to.

Fully Local

No servers. No cloud. No analytics. Everything runs on your machine. Open source and fully auditable.

Quick Start

Three steps to automate logins

Installation
1
Install 1Password CLI
brew install --cask 1password-cli
2
Run VaultRunner setup
npx vaultrunner setup
3
Add MCP to Claude Code
claude mcp add vaultrunner -s user -- npx vaultrunner-mcp
