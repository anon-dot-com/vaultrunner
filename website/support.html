Frequently Asked Questions

How do I install the 1Password CLI?

On macOS, run brew install --cask 1password-cli. For other platforms, visit the 1Password CLI downloads page.

Why isn't VaultRunner filling my password?

Make sure 1Password is unlocked and the CLI is authenticated. Run npx vaultrunner status to check your setup.

How do I enable SMS code reading?

On macOS, grant Full Disk Access to your terminal app in System Preferences → Privacy & Security. This allows reading from the Messages database.

Is my data sent to any servers?

No. VaultRunner runs entirely locally. Your credentials never leave your machine and are never sent to any external servers.

Does Claude see my passwords?

Claude receives credentials from VaultRunner via MCP and uses Claude for Chrome to fill forms. All processing happens locally on your machine—nothing is sent to external servers.

How do I update VaultRunner?

The MCP server updates automatically via npx. Run npx vaultrunner@latest setup to ensure you have the latest version.
