FAQ

How do I install 1Password CLI?

On macOS: brew install --cask 1password-cli

Password not filling?

Make sure 1Password is unlocked. Run npx vaultrunner status to check.

How do I enable SMS codes?

Grant Full Disk Access to your terminal in System Preferences → Privacy & Security.

Is my data sent anywhere?

No. Everything runs locally. Credentials never leave your machine.

Does Claude see passwords?

Claude uses VaultRunner via MCP to fill forms locally. Nothing goes to external servers.

How do I update?

Run npx vaultrunner@latest setup for the latest version.
